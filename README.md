# IAV DESeq2 Analysis

Programa creado en Python para identificar genes expresados diferencialmente con resultados de DESeq2 en una infección por Influenza A.

El programa lee un archivo TSV con genes, valores de log2 Fold Change y adjusted p-value, identifica genes significativamente regulados y genera un archivo de salida con su clasificación-

Ademas, permite configurar los thresholds de significancia en la línea de comandos.
